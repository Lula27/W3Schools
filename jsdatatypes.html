The Concept of Data Types

Different sequences produce different results:

Primitive Data: single data value with no additional properties and methods.

typeof operator can return one of these primitive types:

  • string
  • number
  • boolean
  • null
  • undefined

Complex Data:

typeof operator returns two complex types:

  • function
  • object

Examples:

Undefined

In JavaScript, a variable without a value, has the value undefined.

Wait! What?!

Surprisingly, in JavaScript, the data type of null is an object.
