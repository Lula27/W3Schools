JavaScript Display Possibilities

JS displays data in different ways:

  • innerHTML- write into an HTML element *access HTML element with document.getElementById(id) method; id attribute defines HTML element; innerHTML property defines HTML content

    • Example

  • window.alert()- write into an alert box
  • console.log()- write into browser console *useful for debugging*
  • document.write- write into HTML output; *Using document.write() after an HTML document is fully loaded will delete all existing HTML (useful for testing)
