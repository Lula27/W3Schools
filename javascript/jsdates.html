JavaScript Dates

The Date object lets you work with dates (years, months, days, hours, minutes, seconds, and milliseconds)

JavaScript Date Formats

  • String-Mon May 08 2017 09:44:19 GMT-0400 (Eastern Daylight Time)
  • Number-1494251059068   Specifies number of milliseconds since January 1, 1970, 00:00:00.
