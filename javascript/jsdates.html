JavaScript Dates

The Date object lets you work with dates (years, months, days, hours, minutes, seconds, and milliseconds)

Date objects are static. The computer time is ticking, but date objects, once created, are not.

If you don't specify a time zone, JavaScript will use the browser's time zone.

JavaScript Date Formats

  • String-Mon May 08 2017 09:44:19 GMT-0400 (Eastern Daylight Time)
  • Number-1494251059068   Specifies number of milliseconds since January 1, 1970, 00:00:00.

Creating Date Objects

Date consists of a year, month, day, hour, minute, second and milliseconds.

Create date objects with the new Date( ) constructor.

Here are the 4 ways of initiating a date:

  • new Date( )
  • new Date(milliseconds)
  • new Date(dateString)
  • new Date(year, month, day, hours, minutes, seconds, milliseconds)

    Date Methods

    After creating a Date object, use various methods to operate on it.

    Date methods let you get and set the year, month, day, hour, minute, second and milliseconds of objects, using their local time or UTC (universal, or GMT) time.
