JavaScript Arrays

Use JavaScript arrays to store multiple values in a single variable.

Example

var cars = ["Saab", "Volvo", "BMW"];

Arrays use numbered indexes.

Objects use named indexes.

***Arrays are a special kind of objects, with numbered indexes.***

Arrays with named indexes are called associative arrays (hashes).

When to use Arrays vs Objects

  • JS doesn't support associative arrays.
  • Use objects when you want to element name to be strings (text)
  • Use arrays when you want the element names to be numbers
