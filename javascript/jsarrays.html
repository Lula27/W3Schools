JavaScript Arrays

Use JavaScript arrays to store multiple values in a single variable.

Example:

var cars = ["Saab", "Volvo", "BMW"];

Adding Array Elements

A. (easiest) Push Method

var fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.push("Lemon");

B. Length Property

fruits[fruits.length] = "Lemon";
