JavaScript Math Object

The JavaScript Math object allows you to perform mathematical tasks on numbers.

Math object has no constructor, unlike other global objects. Its methods and properties are static.

In other words, all methods and properties (constants) can be used without first creating a Math object.

Example: Math.PI

Math.round( )

Math.round(x) returns the value of x rounded to its nearest integer:

Math.round(4.7);

Math.round(4.4);

Math.pow( )

Math.pow(x,y) returns the value of x to the power of y:

Math.pow(8,2);

Math.sqrt( )

Math.sqrt(x) returns the square root of x:

Math.sqrt(64);

Math.abs( )

Math.abs(x) returns the absolute (positive) value of x:

Math.abs(-4.7);

Math.ceil( )

Math.ceil(x) returns the value of x rounded up to its nearest integer:

Math.floor( )

Math.floor(x) returns the value of x rounded down to its nearest integer:

Math.sin( )

Math.sin(x) returns the sine (a value between -1 and 1) of the angle x (given in radians).

Use following formula to convert degrees to radians:

Angle in radians = Angle in degrees x PI / 180

Example: Math.sin(90 * Math.PI / 180);

Math.cos( )

Math.cos(x) returns the cosine (btw -1 and 1) of the angle x (given in radians). Use following formula to convert degrees to radians:

Angle in radians = Angle in degrees x PI / 180

Math.min( ) & Math.max( )

Math.min( ) and Math.max( ) can be used to find the lowest/highest value in a list of arguments:

Example:Math.min(0, 150, 30, 20, -8, -200);

Example:Math.max(0, 150, 30, 20, -8, -200);

Math.random( )

Math.random( ) returns a random number between 0 (inclusive), and 1 (exclusive)

Math Properties (Exercises)

JS provides 8 mathematical constants that can be accessed with the Math object:

Math.E // returns Euler's number

Math.PI // returns PI

Math.SQRT2 // returns the square root of 2

Math.SQRT1_2 // returns the square root of 1/2

Math.LN2 // returns the natural logarithm of 2

Math.LN10 // returns the natural logarithm of 10

Math.Log2E //returns the natural logarithm of E

Math.LOG10E // returns base 10 logarithm of E
