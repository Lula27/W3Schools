JavaScript Objects

Te recuerdas que...variables are containers for data values. Well, objects are also variables. But objects can contain many values.

JavaScript objects are containers for named values called properties and/or methods.

Example:

Object Properties

The name:value pairs (in JS objects) are called properties.

Accessing Object Properties

How? Two ways...

A.   objectName.propertyName

B.   objectName["propertyName]

Object Methods

A Method is a function inside an object.

Methods are actions that can be performed on objects.

Example...function() {return this.firstName + " " + this.lastName;}

Accessing Object Methods

How? ... Use the following syntax:

objectName.methodName()
