JavaScript Scope

In JS, scope is the set of variables, objects, and functions you have access to.

A. Local JavaScript Variables

Local Variables- variables declared within a function

Local variables have local scope: They can only be accessed/recognized within the function.

Because of ^ variables with the same name can be used in different functions.

Local variables are created when a function starts and deleted with the function is completed.

B. Global JavaScript Variables

Global Variables- declared outside a function

A global variable has local scope: All scripts and functions on a web page can access it.

C. Automatically Global

If you assign a value to a variable that has not been declared, it will automatically become a GLOBAL variable.

Global Variables in HTML

JavaScript - global scope = complete JavaScript enviornment

HTML - global scope = window object

All global variables belong to the window object.
