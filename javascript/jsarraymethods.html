JavaScript Array Methods

The strength of JavaScript lies in the array methods.

Converting Arrays to Strings

The JavaScript method toString( ) converts an array to a string of (comma separated) array values.

Example: fruits.toString();

The join( ) method also joins all array elements into a string.

Similar to toString( ), but you can also specify and separator:

Example: .join(" * ");

Shifting Elements

Shifting is similar to popping but instead of working on the last element, it works on the first.

The shift( ) method removes the first element and "shifts"all other elements to a lower index.

Starting array: fruits = ["Bannana", "Orange", "Apple", "Mango"];

fruits.shift( );

unshift( ) method adds a new element to array (at begining), and "unshifts" older elements:

Starting array: var fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.unshift("Lemon");

Changing Elements

Array elements are accessed using their index number:

Starting array: var fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits[0] = "Kiwi";

Length Property

Also, add element by using the length property:

Staring array: fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits[fruits.lenght] = "Kiwi";

Deleting Elements

Since JavaScript arrays are objects, elements can be deleted by using the JavaScript operator delete:

Staring array: var fruits = ["Banana", "Orange", "Apple", "Mango"];

delete fruits[0];

Splicing an Array

Use splice( ) method to add new items to an array:

Starting Array: var fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.splice(2, 0, "Lemon", "Kiwi");

  • first parameter (2) defines the position where new elements should be added (spliced in
  • second parameter (0) defines how many elements should be removed
  • rest of the parameters ("Lemon" , "Kiwi") define the new elements to be added

Using splice( ) to Remove Elements

Tweek parameters to use splice( ) to remove elements w/o leaving "holes" in array:

Starting array: var fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.splice(0, 1); -removes first element of fruits

  • first parameter (0) defines the position where new elements should be added (spliced in)
  • second parameter (1) defines how many elements should be removed
  • omitted rest of parameters - won't add any new elements

Merging (Concatenating) Arrays

The concat( ) method creates a new array by merging (concatenating) existing arrays:

Example 1 (Merging Two Arrays)

var myGirls = ["Cecilie", "Lone"];

var myBoys = ["Emil", "Tobias","Linus"];

var myChildren = myGirls.concat(myBoys);

Example 1 (Merging Three Arrays)

var arr1 = ["Cecilie", "Lone"];

var arr2 = ["Emil", "Tobias","Linus"];

var arr3 = ["Robin", "Morgan"];

var misHijos = arr1.concat(arr2, arr3); // Concatenates arr1 with arr2 and arr3

Example (Merging an Array with Values)

var array1 = ["Cecilie", "Lone"];

var myBabies = arr1.concat(["Emil", "Tobias","Linus"]);
