JavaScript Sorting Arrays

Sorting an Array

The sort( ) method sorts an array alphabetically:

By default, sorts values as strings.

var fruits = ["Banana", "Orange", "Apple", "Mango"];

Reversing an Array

The reverse( ) method reverses the elements in an array.

May be used to sort array in decending order:

Numeric Sort

Since sort works by default for strings, create a function to sort numeric values.

Compare function:

A. Ascending Order

var points = [40, 100, 1, 5, 25, 10];

points.sort(function(a, b){return a - b});

num = [-17, 0, 500, -77, 13, 5]

B. Descending Order

points.sort(function(a, b){return b - a});

num = [-17, 0, 500, -77, 13, 5]


Experimential Practice

Sorting an Array in Random Order

var points = [40, 100, 1, 5, 25, 10];

points.sort(function(a, b){return 0.5 - Math.random()});

Finding the Highest & Lowest Values

var numeros = [40, 100, 1, 5, 25, 10];

A. Highest

points.sort(function(a, b){return b - a});

document.getElementById("hi").innerHTML = numeros[0];

B. Lowest

points.sort(function(a, b){return a - b});

document.getElementById("low").innerHTML = numeros[0];
