JavaScript If...Else Statements

Conditional statements used to perform different actions based on specific conditions.

Conditional Statements

Use conditional statements to perform different actions for different decisions!

JS has following conditional statements:

  • if- specify a block of code to be executed if a specified condition is true
  • else- specify block of code to be executed if same condition is false
  • else if- specify new condition to test, if first condition is false
  • switch- specify many alternative blocks of code to be executed
