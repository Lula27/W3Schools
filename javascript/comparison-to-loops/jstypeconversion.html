JavaScript Type Conversion

Number() converts to a Number, String() converts to a String, Boolean() converts to a Boolean.

In JS, there are 5 different data types that contain values:

  • string
  • number
  • boolean
  • object
  • function

3 types of objects

  • object
  • date
  • array

2 data types that cannot contain values:

  • null
  • undefined

The typeof Operator

Use typeof operator to find the data type of a JS variable

var name = typeof "John"

typeof NaN

typeof {name: 'John', age: 34}
