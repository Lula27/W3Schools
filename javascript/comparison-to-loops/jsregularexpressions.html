What is a Regular Expression?

  • A regular expression is a sequence of characters that forms a search pattern.
  • Can be a single character, or a complicated pattern.
  • Can be used to perform all types of text search and text replace operations

Syntax

/pattern/modifiers;

Example

var patt = /w3schools/i;

Using String Methods

In JavaScript, regular expressions are often used with the two string methods: search( ) and replace( ).

    A. search( ) method- uses an expression to search for a match, and returns the position of the match.

    Using String search() With a Regular Expression

    var n = str.search(/w3schools/i);

    Using String search( ) with String

    var n = str.search("W3Schools");

    B. replace( ) method- returns a modified string where the pattern is replaced.
