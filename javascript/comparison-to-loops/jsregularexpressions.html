What is a Regular Expression?

  • A regular expression is a sequence of characters that forms a search pattern.
  • Can be a single character, or a complicated pattern.
  • Can be used to perform all types of text search and text replace operations

    • Regular expressions can make your search more powerful (case insensitive for example)...

Syntax

/pattern/modifiers;

Example

var patt = /w3schools/i;

Using String Methods

In JavaScript, regular expressions are often used with the two string methods: search( ) and replace( ).

    A. search( ) method- uses an expression to search for a match, and returns the position of the match.

    Using String search() With a Regular Expression

    var n = str.search(/w3schools/i);

    Using String search( ) with String

    var n = str.search("W3Schools");

    B. replace( ) method- returns a modified string where the pattern is replaced.

    String replace() with a Regular Expression

    var str = "Visit Microsoft!";

    var res = str.replace(/microsoft/i, "W3Schools");

    Using String replace() With a String

    var str = "Visit Microsoft!";

    var res = str.replace("Microsoft", "W3Schools");

Using test() Method

test( ) method = RegExp expression method

Searches string for a pattern and returns true or false, depending on the result.

The following example searches a string for the character "e":

var patt = /e/;

patt.test("The best things in life are free!");

Using exec( )

exc( ) method = RegExp expression method

Searches string for a specified pattern, and returns the found text. If no match is found, it returns null.

Example: find e again!

/e/.exec("The best things in life are free!");
